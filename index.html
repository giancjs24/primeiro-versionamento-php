Primeiro site

Estamos tendo o primeiro contato com HTML, CSS e JS. Sao as chamdas tecnologias frontend, que é basicamente como vai ser a interface das aplicaçoes, dos sites ou de qualquer coisa que o usuário vai ver. E estamos criando nosso primeiro exemplo para entender como funciona essas tecnologias. HTML é uma linguagem de marcaçao, ou seja, eu posso inserir um texto(Assim como esse que estou digitando agora), um titulo, imagens, listas, tabelas e mais. CSS diz respeito a estilizaçao do site: cores, formas, divisao dos elementos na tela e tudo que esta relaciona a design. Javascript é a linguagem de programaçao do frontend, com ele sim iremos programar de fato, porém, mais para frente ainda nessa Unidade curricular.
